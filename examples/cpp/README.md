# :croissant: C++ Example 

This example shows how to use FracGM and QGM in C++.

```shell
cd examples/cpp
```

## :gear: Build

```shell
mkdir build
cd build
cmake .. 
make
```

## :checkered_flag: Run

```shell
./registration_example
```

This main function will read two point clouds from the [`data`](../data) folder and solve the registration problem.
